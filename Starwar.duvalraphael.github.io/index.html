Il y a longtemps dans une galaxie lointaine.

Un apprenant devait faire un menu animé Star War mais il se trompa de film.

Il décida qu'il fallait un apprenant pour les surpasser tous, un apprenant pour les gouverner tous. Mais la lutte pour le trône de becode était compliquée. De nombreux apprenants avaient plus de facilité que lui. Voyant les pertes engendraient. Les anciennes Némésis et notre héros décidèrent de s'associer et de gouverner ensemble pour un code plus fort, ensemble pour un code plus indenté.
